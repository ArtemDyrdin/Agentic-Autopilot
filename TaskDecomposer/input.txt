Проект: Платформа онлайн-обучения для корпоративных клиентов

1. Общая цель проекта

Разработать веб-платформу для корпоративного обучения сотрудников.
Платформа должна позволять компаниям создавать учебные программы, назначать курсы сотрудникам и отслеживать прогресс.

Система должна поддерживать многоклиентскую архитектуру (multi-tenant), где каждая компания работает в изолированном пространстве.

---

2. Роли пользователей

2.1 Суперадминистратор платформы
- Управляет корпоративными аккаунтами
- Просматривает общую статистику использования
- Управляет тарифами

2.2 Администратор компании
- Создает сотрудников
- Создает учебные программы
- Назначает курсы
- Просматривает отчеты

2.3 Сотрудник
- Проходит назначенные курсы
- Сдает тесты
- Просматривает свой прогресс

---

3. Функциональные требования

3.1 Управление компаниями
- Регистрация компании
- Выбор тарифного плана
- Управление подпиской
- Ограничение количества пользователей по тарифу

3.2 Управление пользователями
- Приглашение сотрудников по email
- Назначение ролей
- Деактивация пользователя

3.3 Управление курсами
- Создание курса
- Добавление модулей
- Добавление видео, текста и файлов
- Создание тестов с вопросами
- Настройка минимального проходного балла

3.4 Прохождение обучения
- Просмотр контента
- Сохранение прогресса
- Прохождение теста
- Ограничение количества попыток
- Автоматическое начисление сертификата

3.5 Отчеты и аналитика
- Отчет по прохождению курса
- Отчет по сотруднику
- Экспорт в CSV
- Дашборд с ключевыми метриками

---

4. Нефункциональные требования

- Время отклика API не более 300 мс
- Поддержка 10 000 одновременных пользователей
- Данные разных компаний должны быть изолированы
- Логи должны храниться минимум 1 год
- Резервное копирование ежедневно

---

5. Интеграции

- Интеграция с SSO (OAuth2 / SAML)
- Интеграция с платежным провайдером
- Отправка email-уведомлений

---

6. Ограничения

- Веб-приложение (desktop-first)
- Backend — REST API
- Аутентификация через JWT
- Хранение видео в облачном хранилище

---

7. Метрики успеха

- 50 корпоративных клиентов в первый год
- Средний completion rate курсов не менее 70%
- Удержание клиентов > 80%
